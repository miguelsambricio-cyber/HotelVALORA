## 2 · Definitive 5-layer architecture

### 2.1 Conceptual model

| Layer | Definition | Storage | Who creates | Who sees |
|---|---|---|---|---|
| **Showcase** | Official HotelVALORA demo reports · operator-curated · used for sales/demos/investor pitches | `hotel_report_library` rows with `report_origin = 'showcase'` | Operator (admin) | Everyone (anon · auth · admin) |
| **Community** | Public reports generated by users that they chose to share publicly | `hotel_report_library` rows with `report_origin = 'community'` (post-MVP) | Authenticated users | Everyone |
| **Top Promote** | A FLAG on any row · paid placement that surfaces a report on top of lists/maps + reveals contact channel | `hotel_report_library.is_top_promote = true` + `contact_visible = true` | Operator (admin) sets · monetised | Filtered surfaces (top-list/map prioritise these) |
| **Favorites** | Per-user private collection · star-marker on any visible row | New table `report_favourites (user_id, library_id, created_at)` | Authenticated users (self) | Only the owning user |
| **Institutional Library** | Full audit / QA / admin view of every row regardless of origin · all 224 bulk_seed + showcase + community + engine_render rows | Admin-only `/user/admin/library` view (new) reading raw `hotel_report_library` with no filters | n/a (read-only) | Admin only |

### 2.2 Tier badge vs origin · clarification

`tier_badge` (Free · Pro · Premium) is **orthogonal** to `report_origin` (showcase · community · engine_render · …):

- A `showcase` row can be `Premium` (Mandarin Ritz · full content) or `Free` (entry-level demo).
- A `community` user-generated report can be `Free` (default) or upgraded to `Pro`/`Premium` by the operator.
- `engine_render` rows are typically `Free` until promoted.

`is_top_promote` is **orthogonal** to BOTH:
- A `Free` row can be top-promoted (paid placement on lower-tier slot).
- A `Premium` row may or may not be top-promoted.

### 2.3 Schema delta · migration 0027

```sql
-- Add origin classifier (replaces the bulk_seed-vs-real distinction)
ALTER TABLE public.hotel_report_library
  ADD COLUMN report_origin text NOT NULL DEFAULT 'engine_render'
  CHECK (report_origin IN (
    'engine_render',  -- user opened /report/* in a browser
    'showcase',       -- operator-curated official demo
    'community',      -- authenticated user shared a generated report publicly
    'bulk_seed',      -- automated population (the 223 current rows)
    'manual_seed',    -- single SQL INSERT for testing (the 1 Mandarin row)
    'imported',       -- ingested from external source (CoStar CSV · future)
    'migrated'        -- moved from valuations or other legacy table
  ));

-- Tier badge (orthogonal to origin)
ALTER TABLE public.hotel_report_library
  ADD COLUMN tier_badge text NOT NULL DEFAULT 'free'
  CHECK (tier_badge IN ('free', 'pro', 'premium'));

-- Top promote flag + contact visibility
ALTER TABLE public.hotel_report_library
  ADD COLUMN is_top_promote boolean NOT NULL DEFAULT false;
ALTER TABLE public.hotel_report_library
  ADD COLUMN contact_visible boolean NOT NULL DEFAULT false;
ALTER TABLE public.hotel_report_library
  ADD COLUMN contact_info jsonb;

-- Showcase ordering + last-real-render
ALTER TABLE public.hotel_report_library
  ADD COLUMN showcase_priority integer NOT NULL DEFAULT 0;
ALTER TABLE public.hotel_report_library
  ADD COLUMN last_operator_render_at timestamptz;

-- Indexes for the filtered surfaces
CREATE INDEX hotel_report_library_origin_idx
  ON public.hotel_report_library (report_origin);
CREATE INDEX hotel_report_library_promote_idx
  ON public.hotel_report_library (is_top_promote, showcase_priority DESC)
  WHERE is_top_promote = true;
CREATE INDEX hotel_report_library_showcase_idx
  ON public.hotel_report_library (report_origin, showcase_priority DESC)
  WHERE report_origin IN ('showcase', 'community');
```

### 2.4 Schema delta · migration 0028 · per-user favourites

```sql
CREATE TABLE IF NOT EXISTS public.report_favourites (
  user_id    uuid NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  library_id uuid NOT NULL REFERENCES public.hotel_report_library(id) ON DELETE CASCADE,
  created_at timestamptz NOT NULL DEFAULT NOW(),
  notes      text,
  PRIMARY KEY (user_id, library_id)
);
ALTER TABLE public.report_favourites ENABLE ROW LEVEL SECURITY;

-- Each user reads/writes only their own rows
CREATE POLICY report_favourites_self_select ON public.report_favourites
  FOR SELECT TO authenticated USING (user_id = auth.uid());
CREATE POLICY report_favourites_self_insert ON public.report_favourites
  FOR INSERT TO authenticated WITH CHECK (user_id = auth.uid());
CREATE POLICY report_favourites_self_delete ON public.report_favourites
  FOR DELETE TO authenticated USING (user_id = auth.uid());
```
